Delivery Mode: How do we handle this?

The user must choose their workflow. Direct = buy & deliver immediately. Carrier = stockpile on FC first.

A

Mode Toggle

Two-state switch at top: "Direct Delivery" vs "Carrier Mode". Stays visible. Simple but requires user to consciously choose.

B

Auto-Detect

System detects mode from cargo state. If Fleet Carrier cargo exists → carrier mode. If only ship cargo → direct. Less clicks, more implicit.

C

Unified View

Show all three locations (station, carrier, construction) in one view. User decides sequence. Most flexible but complex.

D

Phase-Based

Each phase has its own mode settings. "Scanning" → add sites. "Collection" → choose delivery method. "Done" → summary. Guided flow.
